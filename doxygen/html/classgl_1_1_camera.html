Detailed Description

Class to manage and control a virtual camera the world space. After setting camera properties and a viewing frustum, users can retrieve related view and projection matrices. If the methods for the matrices are called without setting, those return 4x4 identity matrix. This class would be extended to support movements like work around and looking around, such as pitch and yaw.
