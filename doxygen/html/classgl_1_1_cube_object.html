Detailed Description

Class to manage and draw a cube object. This class inherits the Triangle class because a cube is a composition of multiple triangles. So, all implementation is same with triangle class except how to draw for the next frame. In this version, the internal implementation has upgraded to support multiple objects and Element Buffer Objects.
